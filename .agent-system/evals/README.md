# Evaluations

Store durable cases used to compare baseline and skill-assisted behavior.

A useful eval is representative of real repository work, specific enough to
score, isolated from secrets and production data, resistant to superficial
wording, and explicit about forbidden behavior.

Use `.agent-system/contracts/eval-case.md`.

Candidate-specific evals live beside the candidate. After promotion, copy stable
regression cases here under a directory named after the skill.
