# Context Packet Contract

```markdown
# Context Packet

## Task summary
One precise paragraph.

## Acceptance evidence
What test, command, diff property, or runtime observation proves completion?

## Governing instructions
- Relevant project profile sections
- Relevant nested instructions
- Applicable skills

## Execution path
| Step | File/symbol | Evidence |
|---|---|---|

## Existing behavior
Observed facts only.

## Working set
- Files likely to change
- Tests likely to change
- Read-only references

## Constraints and invariants
- Compatibility
- Data integrity
- Security
- Performance
- Style

## Hypotheses
| Hypothesis | Supporting evidence | Disconfirming check |
|---|---|---|

## Unknowns
- Resolvable through investigation
- Requires user or external access

## Verification ladder
1. Targeted check
2. Package check
3. Broader check
4. Runtime check

## Context exclusions
Files or topics intentionally not loaded and why.
```
