# Skill Proposal Contract

Save as `proposal.md` beside a candidate `SKILL.md`.

```markdown
# Skill proposal: <name>

## Problem
What repeated or costly failure does this solve?

## Evidence of recurrence
- Run/task reference 1
- Run/task reference 2
- Run/task reference 3

## Intended trigger
Tasks that should load the skill.

## Exclusions
Tasks that must not load it.

## Inputs
Required repository state or user information.

## Outputs
Expected artifacts or decisions.

## Baseline
How the workflow behaves without the skill.

## Expected improvement
Reliability, context, time, verification, or safety improvement.

## Risks
False trigger, stale assumptions, excess permissions, or overfitting.

## Rollback
How to disable or remove the skill.

## Evaluation status
candidate | passed | rejected | promoted
```
