# Task Contract

Use this shape to frame non-trivial work.

```markdown
# Task

## Outcome
What observable result must exist?

## Acceptance criteria
- [ ] Concrete behavior or artifact
- [ ] Required compatibility
- [ ] Required verification

## Non-goals
- Explicitly excluded work

## Constraints
- Repository, time, dependency, security, or API constraints

## Risk
small | medium | high

Reason:

## Assumptions
- Assumption and evidence

## Blocking unknowns
- Unknown that cannot be resolved from repository evidence
```
